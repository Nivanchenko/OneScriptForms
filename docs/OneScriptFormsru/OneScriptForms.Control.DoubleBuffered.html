ЭлементУправления.ДвойнаяБуферизация (Control.DoubleBuffered) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, должна ли поверхность этого элемента управления перерисовываться с помощью дополнительного буфера, чтобы уменьшить или предотвратить мерцание.

Значение Истина, если поверхность элемента управления должна перерисовываться с помощью двойной буферизации, Ложь - в противном случае.

Примечание

Пример

Форма1.ДвойнаяБуферизация = Флажок1.Помечен;
