Варианты установки обработчика события

Начиная с версии 1.1.0.0 расширилась возможность установки обработчика события для объектов.

В версии 1.0.0.0 свойство объекта, указывающее на процедуру или функцию с кодом обработки события, имело строковый тип. Начиная с версии 1.1.0.0 свойству возможно задать также типы "Действие" или "СловарнаяЗапись". Рассмотрим это на примерах.
