АргументыСобытия.Параметр (EventArgs.Parameter) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает объект, заданный при установке обработчика события.

Примечание

Если обработчик события задан объектом с типом значения "Действие", тогда Параметр - это экземпляр типа "Действие".

Если обработчик события задан объектом с типом значения "СловарнаяЗапись", тогда Параметр - это ключ экземпляра типа "СловарнаяЗапись".

Если обработчик события задан типом значения "Строка", тогда Параметр будет иметь значение Неопределено.

Пример

Параметр = Ф.АргументыСобытия.Параметр;
